# Weather Data Analysis Project

## Project Overview

This project collects weather data using Weather API and analyzes the
data to create an interactive Power BI dashboard.

## Tools Used

-   Weather API
-   Power BI
-   Power Query

## Project Process

1.  Fetch weather data using API.
2.  Clean and transform data.
3.  Analyze weather information.
4.  Create an interactive dashboard using Power BI.

## Dashboard Features

-   Temperature Analysis
-   Humidity Analysis
-   Wind Speed Analysis
-   Weather Condition Analysis

## Project Files

-   Power BI Dashboard (.pbix)
-   Dashboard Screenshots

## Conclusion

This project helps to understand weather patterns and visualize weather
data using Power BI.
